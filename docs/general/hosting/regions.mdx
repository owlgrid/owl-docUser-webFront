# Regions

OwlGrid Cloud features for now hosting only in France (based on AWS datacenter `eu-west-3`, located in Paris).

We plan to add the possibility to work with multi-regions in the future. If you have a need, please contact us. You can also check [our roadmap](/docs/general/introduction/roadmap) to know our next steps.
