# Introduction

**Templates** are resources configuration that can be deployed to create new resources... or just to get inspiration.

Every [resource](configuration-files) configuration can be describe in a configuration file. These configurations files allow to create resources in a repeatable way. They can be used to create resources in a new environment, to update existing resources or to create a backup of existing resources.

## Publish a template

Templates are hosted in the [templates repository](https://github.com/owlgrid/templates). You can publish your own templates in the matching folder by creating a pull request.

Please follow the [template contribution guidelines](https://github.com/owlgrid/templates/blob/main/CONTRIBUTING.md) before submitting your pull request.

:::info

Once your pull request is merged, your template will be publicly available in the [template store](https://owlgrid.app/templates). **Check that the template doesn't contain any sensitive information (passwords, tokens, etc.)**

:::
