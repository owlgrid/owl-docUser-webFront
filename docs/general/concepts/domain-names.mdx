# Domain names

By default, a [workspace](/docs/general/concepts/workspaces) is accessible at `https://<workspace-id>.owlgrid.app`. 

It is also possible to use a custom domain name the workspace. This is useful if you want to use your own domain name, or if you want to use a subdomain of your existing domain name.
