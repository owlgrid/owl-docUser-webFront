# Actions

Les actions définissent des choses qui peuvent être effectuées sur une ressource. Un droit consiste en une action appliquée à un périmètre.

Des ressources sont créées automatiquement par OwlGrid en fonction des ressources (ex: create, delete, etc.). Il est possible de créer des actions supplémentaires. TODO -> vraiment utile ?

## Caractéristiques des actions

### Données par défaut

Une action possède une donnée par défaut : son identifiant.

L'identifiant est un verbe. Par exemple : `create`, `update`, `delete`, `read`, `assignRight`, etc.

L'action `admin` permet d'effectuer toutes les actions.
