# Introduction

The **controller** is a special resource that manages a [space](/docs/general/concets/spaces). It is responsible for the following:
- Manage the space identity (id, network, etc.);
- List available resources of the space;
- Attach or detach resources;
- Manage the space configuration:
    - Metadata: more information [here](/docs/general/concepts/metadata);
