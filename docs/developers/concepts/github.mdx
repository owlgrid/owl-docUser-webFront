# Github

L'organisation Github [Owlgrid](https://github.com/owlgrid) contient tous les repositories d'OwlGrid. La nomenclature des repositories est définie dans le Readme d'accueil du repository.

## Types de repositories

Cf. [Github Owlgrid](https://github.com/owlgrid).

### App repositories

Les dépôts d’application sont des dépôts git hébergés sur Github qui contiennent la configuration et le code d’un workspace. La relation entre un workspace et un dépôt d’application est un à un.

Les app repositories contiennent de la **configuration** et du **code** qui permettent de définir le comportement des instances de ressources. Ce couple (configuration, code) est défini par les utilisateurs de la plateforme.

La configuration et le code sont hébergés sous forme de fichiers JSON (configuration) et Typescript et plus (code) dans des dépôts d’application (appelés couramment app repositories).
