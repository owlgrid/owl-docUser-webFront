# Domain names

By default, a [space](/docs/concepts/spaces) is accessible at `https://<space-id>.owlgrid.app`. 

It is also possible to use a custom domain name the space. This is useful if you want to use your own domain name, or if you want to use a subdomain of your existing domain name.
